Quick Review for MadLibs
Mad libs is a game where you answer the series of questions to user and based on the input entered by the user. The input is used to fill in the blanks in a story that we already have.After all the input is received we will create a story which is based on the users input. For this game, we will give you a form on right side of the page and user should fill the form based on the question. All the fields are required so you were not able to skip any questions. If you try to skip it wont allow you to generate the madlibs.
This Madlibs is based on the famous Artist in the art history. Everytime you generate the Madlibs you will generate a story of different artist.You are not able to chose which artist you want because if you know the story then it wont be fun. So the game is created such that the random story is generated every time you play it.
